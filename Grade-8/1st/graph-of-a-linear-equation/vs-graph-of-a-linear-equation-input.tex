\begin{center}
\textbf{Graph of a Linear Equation
}
\end{center}

\vspce

Graphing linear equations can be done using any of the four methods: 
\begin{enumerate}[label = \arabic*. ]
\item \hspce Using two points
\vspce 
\item \hspce Using the x- and  y-intercepts
\vspce 
\item \hspce Using the slope  and the  y-intercept
\vspce 
\item \hspce Using the slope  and a point
 
\end{enumerate}  

%x-intercept:  the  abscissa of  the  coordinates  of  the  point in which  the graph  intersects the  x-axis 

%\vspce

%y-intercept: the ordinate of  the coordinates  of the point in which the graph intersects the y-axis
