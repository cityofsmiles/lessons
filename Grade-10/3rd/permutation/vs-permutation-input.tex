\begin{center}
\textbf{Permutation
}
\end{center}

\vspace*{1ex}

Permutations: the different possible arrangements of a set of objects. \\ The number of permutations of $n$  objects taken $r$ at a time is:  
\[           P(n, r) = \displaystyle \frac{n!}{(n-r)!}, \hspce n\geq r. \]

\vspce 

If $n=r$, then \[
P(n, r) = n! 
\]
